Design a single-page layout with a clear hierarchy. Start with a fixed, 65px tall navbar using a glass-morphism effect. The main Hero section must be full-viewport, featuring a left-aligned text block and a right-aligned visual. Follow this with a multi-row Skills section, a second full-viewport "break" section for a focused visual, and a Projects section built on a responsive 3-card grid. Conclude with a 3-column footer. The entire experience must be backed by a persistent, 3D rotating star canvas fixed at the lowest z-index to provide constant depth.

Establish a dark, space-themed color palette using a deep purple-black (#030014) for the main background. All primary text and headings should stand out using gradients: a light purple-to-blue gradient (#e59cff to #9cb2ff) for welcome text, and a purple-500-to-cyan-500 gradient for section titles. The primary accent color is #7042f8, used for link hovers. Use #2A0E61 for card borders and shadows. Glass morphism elements like the navbar should use a semi-transparent background (#03001427) with a backdrop-filter: blur(). Buttons must have a glowing-inset-shadow effect (#bf97ff3d) that brightens on hover.

Implement a mobile-first responsive design using 768px (md) as the main breakpoint. Below this breakpoint, the navbar must collapse to a logo and hamburger icon (toggling a dropdown), while the Hero and Projects sections stack vertically (flex-col). Above 768px, the navbar must expand to its full horizontal state, the Hero section must switch to a side-by-side (flex-row) layout, and the Projects section must display its cards in a 3-column horizontal grid. Other sections, like Skills and the Footer, should use flex-wrap to stack naturally on smaller screens.
